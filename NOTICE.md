# Notice

This repository is a documentation-focused technical portfolio case study derived from academic coursework in vibroacoustics.

The repository includes curated reports and selected figures. It does not include the original computational source files, raw MATLAB scripts, or full course submission folders. The purpose is to document the modelling assumptions, analytical methods, numerical outputs, and engineering interpretation of the project.

The work was developed as an academic group project. The repository should not be interpreted as a standalone commercial software package or a fully reproducible simulation codebase.

No open-source licence is provided for the reports and figures unless explicitly added later by all authors.
